# Public Assets Folder

Silakan tempatkan Logo K3 untuk Intro di folder ini dengan nama file: **k3-logo.png**

Format yang disarankan:
- Format: PNG (dengan transparansi) atau SVG
- Ukuran: Minimal 400x400px untuk kualitas terbaik
- Background: Transparan (untuk tampilan terbaik)
- Nama file: `k3-logo.png`

Logo akan otomatis digunakan di halaman Intro dengan animasi pop-in saja (tanpa spin).

Jika file tidak ditemukan, logo tidak akan ditampilkan.
